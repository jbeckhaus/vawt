# Vertical Axis Wind turbine 
* uses j wing design to combine lift and drag based wind capture 
* wing shape designed using https://airfoilx.com/ then converted to svg to be used in 3d design 
* designed to be printable on a standard 200 x 200 x 200 sized 3d printer
